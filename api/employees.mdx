## Employees

### List All Employees

<RequestExample>
```bash GET /employee
curl -X GET "${VITE_BASE_URL}/employee" \
  -H "Authorization: Bearer <token>"
```
</RequestExample>

Retrieves a list of all employees.

**Authentication Required:** Yes

### Response

<ResponseField name="data" type="array">
  Array of employee objects
  
  <ResponseField name="id" type="number">
    Employee unique identifier
  </ResponseField>
  
  <ResponseField name="first_name" type="string">
    Employee first name
  </ResponseField>
  
  <ResponseField name="last_name" type="string">
    Employee last name
  </ResponseField>
  
  <ResponseField name="email" type="string">
    Employee email address
  </ResponseField>
  
  <ResponseField name="phone" type="string">
    Employee phone number
  </ResponseField>
  
  <ResponseField name="position" type="string">
    Job position or title
  </ResponseField>
  
  <ResponseField name="department" type="string">
    Department name
  </ResponseField>
  
  <ResponseField name="hire_date" type="string">
    Date of hire
  </ResponseField>
  
  <ResponseField name="status" type="string">
    Employment status (active, inactive)
  </ResponseField>
  
  <ResponseField name="photo_url" type="string">
    URL to employee photo
  </ResponseField>
</ResponseField>

<ResponseExample>

  "data": [
    {

### Create Employee

<RequestExample>
```bash POST /employee
curl -X POST "${VITE_BASE_URL}/employee" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: multipart/form-data" \
  -F "first_name=María" \
  -F "last_name=García" \
  -F "email=maria.garcia@company.com" \
  -F "phone=+52 55 9876 5432" \
  -F "position=Marketing Coordinator" \
  -F "department=Marketing" \
  -F "hire_date=2024-03-01" \
  -F "salary=35000" \
  -F "photo=@/path/to/photo.jpg"
```
</RequestExample>

Creates a new employee record.

**Authentication Required:** Yes  
**Content-Type:** `multipart/form-data`

### Request Body

<ParamField body="first_name" type="string" required>
  Employee first name
</ParamField>

<ParamField body="last_name" type="string" required>
  Employee last name
</ParamField>

<ParamField body="email" type="string" required>
  Employee email address
</ParamField>

<ParamField body="phone" type="string">
  Employee phone number
</ParamField>

<ParamField body="position" type="string" required>
  Job position or title
</ParamField>

<ParamField body="department" type="string">
  Department name
</ParamField>

<ParamField body="hire_date" type="string" required>
  Date of hire (YYYY-MM-DD format)
</ParamField>

<ParamField body="salary" type="number">
  Employee salary
</ParamField>

<ParamField body="status" type="string">
  Employment status (default: "active")
</ParamField>

<ParamField body="photo" type="file">
  Employee photo file
</ParamField>

### Response

<ResponseField name="id" type="number">
  Created employee ID
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseExample>
```json Response
{
  "id": 15,
  "message": "Employee created successfully"
}
```
</ResponseExample>

### Update Employee

<RequestExample>
```bash PUT /employee/{id}
curl -X PUT "${VITE_BASE_URL}/employee/1" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: multipart/form-data" \
  -F "position=Senior Sales Manager" \
  -F "salary=60000"
```
</RequestExample>

Updates an existing employee record.

**Authentication Required:** Yes  
**Content-Type:** `multipart/form-data`

### Path Parameters

<ParamField path="id" type="number" required>
  Employee ID to update
</ParamField>

### Request Body

Same fields as Create Employee (all optional for updates)

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseExample>
```json Response
{
  "message": "Employee updated successfully"
}
```
</ResponseExample>
